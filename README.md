# bilidown-web
## 简介
这是一个前后端分离的项目，借助springboot代理b站官方接口，配合前端页面完成b站视频链接的在线获取。

<img src="https://github.com/Zhouqluo/bilidown-web/blob/main/profile/bilidown-web.jpg">


## 技术栈
前端：vue2 + axios + element-ui 2.15.14

后端：springboot

## 当前版本

V1.5

- bv号，手机端链接解析。
- 视频预览，视频下载与取消，封面图下载。
- 下载进度UI。
